# OPINFINIA - Personal Portfolio Website

---

## Project Overview

**OPINFINIA** is my personal portfolio website. The website showcases my mission to empower African youth through **technology-driven education**, with a focus on students in **rural schools** across the continent.

## Features

### 1. Home Page (`Home.html`)
- Introduction to Lisette Mukiza  
- vision statement  
- Social media links  
- buttons  

### 2. About Page (`About.html`)
- Bio  
- Technical and soft skills section with progress bars    
- Profile image  

### 3. Mission Page (`mission.html`)

## Setup Instructions

1. **Clone or Download the Repository**
 ```
   https://github.com/lisette-lachiever/OPINFINIA.git
````

2. **Ensure File Structure Matches**

   * All CSS files are stored in the `styles/` folder
   * All images are stored in the `assets/` folder
   * HTML files remain in the project root
  

5. **Launch the Website**

   * Open `Home.html` in your browser to preview the site.

## Contact Form Setup

The **contact form** uses the **Web3Forms API** for submissions.
The access key is pre-configured in the HTML, and form data is sent directly to the registered email address.
